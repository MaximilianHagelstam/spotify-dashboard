# Spotify Dashboard

## Prerequisites

- Node.js 14<

## Getting Started

```
git clone https://github.com/MaximilianHagelstam/spotify-dashboard.git
cd spotify-dashboard

npm install
npm run build
npm start
```

## Stack

- Next.js
- Tailwind CSS
