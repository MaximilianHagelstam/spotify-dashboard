# Spotify Dashboard

## Prerequisites

- Node.js 14<
- Yarn

## Getting Started

```
git clone https://github.com/MaximilianHagelstam/spotify-dashboard.git
cd spotify-dashboard

yarn install
yarn build
yarn start
```

## Stack

- Next.js
- Tailwind CSS
